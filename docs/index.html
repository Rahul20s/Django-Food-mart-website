Key Features

👤 User Authentication

🏪 Restaurant Dashboard

Restaurant owners can manage their menu items, view orders, and track inventory

🛒 Food Ordering

Customers can browse food items, place orders, and manage their profiles

Getting Started

🚀 Local Development

git clone https://github.com/Rahul20s/Django-Food-mart-website.git
cd Django-Food-mart-website
pip install django django-extensions pillow
python manage.py migrate
python manage.py runserver

🌐 Deployment

This Django app requires a Python-capable hosting platform:

  • 🔗 Render
  • 🔗 Railway
  • 🔗 Fly.io
  • 🔗 Heroku
